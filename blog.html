03/22/2019

Today I was able to learn a bit more about how the internet works through modules in the course I am currently enrolled in. It was exciting and enlightening. I found that summarizing what I learned after watching each module helps me to learn best! I have provided the notes of what I learned below:

Browsing the Internet

What actually happens when I type in an IP address:

  • The browser sends off a request to the Internet Service Provider asking “who this is”. The ISP then sends off a request to the Domain Name Server which acts like a phone book and then sends the IP address for the website. This is sent back to the browser.
  • The browser then send off a request to the Google Servers that it now knows how to locate with the IP address asking for the HTML,CSS, and Javascript for the webpage. This is sent back to the browser.
  • The browser then interprets the contents of the HTML, CSS, and JavaScript files to display the website.

Breaking Google

I learned how to access the developer tools at any webpage. Once I opened the developer tools where you can view the HTML for the webpage, I was able to change different attributes of the google webpage by making my own modifications to the HTML for the webpage.

A browser is just a piece of software that can receive files from servers and it knows how to read them so that we can view our webpage.

The Internet Backbone

Wow! This was such an interesting module. I didn’t realize how incredible and amazing the internet is! I mean I always knew it was cool that it connects the world but it was this magical mystery that I didn’t think too much into.

So this is what I learned: In order for your browser to contact the ISP and for the ISP to contact the DNS, some incredible connections occur. First, you get WiFi from your router. The router is then connected to the modem which sends off a signal to a tower that connects back to the location of the ISP through a cable. Wow an underground cable. But then the AMAZING part is that then the ISP is connected to the “internet backbone” through a cable.

This is the part that truly amazed me. The internet backbone then connects to locations across the world through wires. Some of these wires run under the sea! Ok, now I want to sing the little mermaid. But seriously that’s amazing!

I will never again complain of how slow the internet is. How does the information travel so fast. Wow.

Oh also: I saw that the internet really just connects a lot of computers that store information. How amazing is this to be a part of?! Ok I’m excited. I’m glad I looked into this mystery of how information travels.
